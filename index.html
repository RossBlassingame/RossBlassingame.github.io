M.A.R.S. - Martian Autonomous Routing System

Worked with NASA JPL as Team Lead on a team of six Computer Science seniors as our Senior Capstone. Given an elevation map of the Martian terrain, our software calculates the ideal path the Rover should take, along with total distance gained, elevation gained, and a 2D heatmap with the ideal routes displayed.

The Legend of Harambe

The Legend of Harambe is a side-scrolling iOS game made using Unity and C# where the user controls an animated gorilla and tries to collect bananas while avoiding obstacles. It was built by me and five other Computer Science students as a group project for the Software Development: Methods and Tools class at CU Boulder. The project won second place out of 20+ total projects.
